# v1 눌림목 모멘텀 전략

최종 업데이트: 2025년

---

## 전략 개요

당일 급등주가 고점(HOD) 도달 후 -2~-5% 눌리는 구간에서
거래량 감소를 확인하고 반등 시 매수하는 전략

---

## 5단계 매매 흐름

종목 스캔 → 급등 확인 → 눌림 감지 → 매수 진입 → 청산

---

## 1단계 — 종목 선별 (키움 조건검색식)

| 조건 | 설정값 |
|------|--------|
| 거래량 | 전일 대비 200% 이상 |
| 상승률 | +3% 이상 |
| 시가 대비 | 현재가 시가 위 유지 |
| 전일 고가 | 현재가가 전일 고가 돌파 |
| 제외 | 관리종목 / 거래정지 / 3,000원 미만 |

스캔 시작: 09:05 (동시호가 노이즈 제거)

---

## 2단계 — 급등 확인 (Python 모니터링)

- 당일 최고가(HOD) 갱신 기록 저장
- HOD 도달 시 거래량 평균 3배 이상 확인
- 이 단계에서 매수 금지 — 눌림 기다림

---

## 3단계 — 눌림목 감지 (Python 모니터링)

| 조건 | 기준 |
|------|------|
| 눌림 범위 | HOD 대비 -2% ~ -5% |
| 거래량 | 눌림 중 감소 확인 |
| 시가 사수 | 현재가 시가 위에 있어야 함 |

---

## 4단계 — 매수 진입 (REST API 주문)

| 조건 | 기준 |
|------|------|
| 트리거 | 직전 1분봉 대비 +1% 이상 반등 |
| 진입 시간 | 09:05 ~ 10:30 이내만 |
| 1일 한도 | 최대 3~5종목 |

---

## 5단계 — 청산 (자동)

| 구분 | 기준 |
|------|------|
| 손절 | 진입가 대비 -3% 즉시 |
| 1차 익절 | +4% 도달 시 50% 청산 |
| 2차 익절 | 5일선 이탈 시 잔량 청산 |
| 강제 청산 | 15:10 전량 청산 |

---

## 주의사항

- 손절 -3% 자동 설정 없이 실전 투입 금지
- 모의투자 100회 이상 테스트 후 실전 시작
- 10:30 이후 신규 진입 금지
- 하루 최대 손실 -2% 초과 시 당일 매매 중단
