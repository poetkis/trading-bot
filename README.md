# 키움 REST API 자동매매 프로그램

키움증권 REST API를 활용한 단타 자동매매 시스템

---

## 전략

눌림목 모멘텀 전략
당일 급등주가 고점 도달 후 -2~-5% 눌리는 구간에서
거래량 감소 확인 후 반등 시 매수

---

## 폴더 구조

trading-bot/
├── strategy/
│   └── v1_눌림목전략.md
├── backtest/
│   └── 결과기록.md
├── src/
│   └── main.py
└── README.md

---

## 매매 시간

- 스캔 시작: 09:05
- 신규 진입 마감: 10:30
- 강제 청산: 15:10

---

## 손익 기준

| 구분 | 기준 |
|------|------|
| 손절 | -3% 즉시 |
| 1차 익절 | +4% (50% 청산) |
| 2차 익절 | 5일선 이탈 시 잔량 |
| 강제 청산 | 15:10 전량 |

---

## 사용 기술

- Python 3.x
- 키움증권 REST API
